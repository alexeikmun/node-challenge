# youcan-api #
YouCan is a mobile application to archive your goals to stay motivated to accomplish it.


## Coding information ##

#### Start project ####

1. git clone 
2. cd youcan-api
3. npm i
4. npm start


#### Debug on chrome console ####

1. npm run debug
2. open on chrome chrome://inspect/#devices
3. click en remote targets for the app
